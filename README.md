# ChromeTabsManager
Временно помогает решить* проблему Chrome со сплющенными вкладками

Это приложение имитирует вертикальные вкладки, которые есть не у всех, а главно помогает ориентироваться во вкладках пролистывая их превычным образом.
Приложение состоит из двух частей:
 - Расширение для браузера. Ставиться как обычное распакованное расширение. Оно отсылает приложению список вкладок и принимает комманды на активацию вкладки, удаление и т.д.
 - Приложение. Написано быстро и на коленке, но в целом работает. Я его делал для себя, по этому дополнительно выложу код если кто-то захочет доработать.
